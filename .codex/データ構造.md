# データ構造

## GameManager
* players : Player[]
* turn : number
* deck : cardId[]
* previewCard : cardId | null
* state : "introduction" | "playing" | "preview" | "gameover"

## Player
* openCard:cardId | null
* hands : cardId[]
* mana : Mana
* life : number

## Mana
* green:number
* red:number
* blue:number

## Card
* id : string
* name : string
* color : "green" | "red" | "blue"
* gainMana : Mana
* text : string
* isFixed : boolean
